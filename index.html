PSU Admission Tester

A program that tests whether or not a student would be admitted to Portland State University

Download this project as a .zip file Download this project as a tar.gz file

The purpose of this project was to use the information available on Portland State University's website to implement a program that tests whether or not a student would be admitted. Here is a sample of the information:

We were also asked to implement a mechanism to report line coverage which is a way to show how many lines of code were used after an operation. Line coverage can be displayed by clicking "I'm Done!".

The project was created with VB.net. To try it, download the project and open the file "~\Code\WP4\WP4.sln" with Microsoft Visual Studio then compile and run the project by clicking Ctrl+F5.

Most of the code can be found in ~\Code\WP4\WP4\Form1.vb. The assignment file can also be found here.
